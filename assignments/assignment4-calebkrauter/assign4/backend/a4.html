Web Service 1: public etailer site for posting cpus

Service Endpoint HTTP METHOD Function/Expected Output
http://localhost:3000/:cpuBrand/:price GET Retrieve all cpus in cpus table that share the same brand and price. (User searches cpu filtered by brand and price -> consumer)
http://localhost:3000/:cpuBrand/:price/:coreCount GET Retrieve all cpus in cpus table that share the same brand, price, and core count.(User searches cpu filtered by brand and price and core count -> consumer)
http://localhost:3000/:cpuBrand/:price/:coreCount/:serialNumber GET Retrieve all data on cpu a specific cpu brand. (User to search by serial number to review listing.)
http://localhost:3000/:serialNumber POST Insert a new cpu by serial number (User uploads cpu to sell by serial number to help ensure that the unique serial number is listed acurately -> consumer)
http://localhost:3000/:cpuBrand/:price/:coreCount/:serialNumber PUT update an existing cpu by serialNumber (User updates listing by brand, price, corecount and serial number to verify what listing is being updated.)
http://localhost:3000/:cpuBrand/:price/:coreCount/:serialNumber DELETE Delete a cpu by serial number (User deletes listing by serial number -> consumer)
